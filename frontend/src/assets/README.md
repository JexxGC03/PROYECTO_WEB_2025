# Assets del Proyecto

Esta carpeta contiene todas las imágenes utilizadas en el sitio web.

## Logo Principal

### logo.png
**Ubicación de uso:** Header de los portales de administrador y cliente

**Instrucciones para reemplazar:**
1. Guarde su logo como `logo.png` en esta carpeta `/assets`
2. Dimensiones recomendadas: 200x200px (cuadrado)
3. Formato: PNG con fondo transparente (recomendado) o con fondo sólido
4. El logo se mostrará en un círculo de 40x40px con fondo blanco en el header
5. Asegúrese de que el logo sea legible en tamaño pequeño

**Nota:** El logo aparece en:
- Portal de Administración (DashboardAdmin)
- Portal del Cliente (DashboardCliente)

## Cómo reemplazar imágenes

Para reemplazar cualquier imagen:
1. Guarda tu nueva imagen con el mismo nombre
2. Asegúrate de que esté en formato JPG o PNG
3. Mantén las dimensiones proporcionales a las originales para mejor visualización

## Dimensiones recomendadas

- Hero images: 1920x1080px o superior
- Team photos: 800x800px (cuadradas)
- Service images: 1200x800px
- Client logos: 400x200px (formato PNG con fondo transparente recomendado)
